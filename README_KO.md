# ⚡ 전기기능사 CBT 시스템

**전기기능사 국가기술자격증 필기시험 대비 온라인 모의고사 시스템**

Computer Based Test (CBT) 시스템으로 전기기능사 이론 시험을 실전처럼 연습할 수 있습니다.

---

## 📋 최신 업데이트 (2025-11-09)

### 주요 개선사항
- ✅ **채점 결과 모달**: 시험 중 채점하기 버튼으로 중간 점수 확인 가능
- ✅ **오답 노트 개선**: 오답노트 복습 모드에서 정답 맞춘 문제 즉시 제거
- ✅ **카테고리별 오답 분류**: 전기이론/전기기기/전기설비별 색상 구분
- ✅ **시험 세션 자동 저장**: 브라우저 종료 시에도 진행 상황 유지
- ✅ **이전 시험 계속하기**: 홈 화면에서 저장된 시험 바로 이어하기

---

## 🎯 주요 기능

### 1️⃣ 시험 모드 (3가지)
- **랜덤 출제**: 전기이론 20 + 전기기기 20 + 전기설비 20 = 총 60문제
- **카테고리별**: 선택한 카테고리에서 20문제 집중 학습
- **오답 노트**: 틀렸던 문제만 최대 20문제 복습

### 2️⃣ 스마트 오답 노트
- 문제를 틀리면 자동 저장
- 카테고리별 오답 분류 (파란색/보라색/주황색)
- 연속 3회 정답 시 자동 제거
- 오답노트 복습 모드: 정답 맞추면 즉시 제거

### 3️⃣ 시험 기능
- ⏱️ **60분 타이머** - 실전과 동일한 시간 제한
- 📊 **중간 채점** - 시험 중 언제든지 점수 확인 가능
- 💾 **자동 저장** - 진행 중인 시험 자동 저장 (브라우저 종료 시에도 유지)
- 🔄 **이전 시험 계속하기** - 저장된 시험 바로 이어서 풀기
- 📝 **해설 보기** - 문제별 상세 해설 제공

### 4️⃣ 통계 및 분석
- 🎓 **분야별 정답률** - 전기이론, 전기기기, 전기설비 등 카테고리별 통계
- 📈 **시험 기록 관리** - 모든 시험 결과 저장 및 분석
- 📊 **차트 시각화** - Recharts 라이브러리로 시각적 통계 제공

### 5️⃣ 관리자 페이지
- 🔐 **비밀번호**: admin2024
- **문제 관리**: 추가/수정/삭제, 페이지네이션, 일괄 삭제
- **사용자 관리**: 23명 초기 회원 + 추가 회원 관리
- **Google Sheets 동기화**: 구글 시트와 양방향 동기화
- **데이터 백업**: JSON 파일로 내보내기/가져오기

### 6️⃣ 기타 기능
- 📱 **반응형 디자인** - PC, 태블릿, 모바일 완벽 지원
- 🔢 **LaTeX 수식** - 전기 공식 및 수식 렌더링 (`$ ... $` 문법)
- 🎨 **TailwindCSS** - 현대적이고 직관적인 UI 디자인

## 📚 문제 출제 기준

### 전기기능사 필기시험 출제 기준 (2024.1.1 ~ 2026.12.31)

#### 📋 시험 구성
- **총 문제 수**: 60문제 (객관식 4지선다형)
- **시험 시간**: 60분
- **과목별 출제 비율**: 각 20문제씩

#### 📖 1. 전기이론 (20문제)
1. **1.1 전기의 성질과 전하에 의한 전기장**
   - 전기의 본질, 정전기, 콘덴서, 전기장과 전위

2. **1.2 자기의 성질과 전류에 의한 자기장**
   - 자석, 전류에 의한 자기현상, 자기회로

3. **1.3 전자력과 전자유도**
   - 전자력, 전자유도, 상호유도

4. **1.4 직류회로**
   - 전압과 전류, 옴의 법칙, 전기저항

5. **1.5 교류회로**
   - 정현파 교류회로, 3상 교류회로, 비정현파

6. **1.6 전류의 열작용과 화학작용**
   - 줄열, 전력량, 전지

#### ⚙️ 2. 전기기기 (20문제)
1. **2.1 변압기**
   - 구조와 원리, 결선, 병렬운전

2. **2.2 직류기**
   - 직류발전기, 직류전동기, 특성

3. **2.3 유도전동기**
   - 원리와 구조, 속도제어

4. **2.4 동기기**
   - 동기발전기, 병렬운전, 동기전동기

5. **2.5 정류기 및 제어기기**
   - 반도체 소자, 정류회로, 사이리스터

6. **2.6 보호계전기**
   - 종류 및 특성

#### 🔧 3. 전기설비 (20문제)
1. **3.1 배선재료 및 공구**
   - 전선, 케이블, 개폐기, 누전차단기

2. **3.2 전선접속**
   - 접속방법, 슬리브, 단자

3. **3.3 배선설비공사 및 전선허용전류**
   - 전선관, 케이블, 옥내배선, 허용전류

4. **3.4 전선 및 기계기구의 보안공사**
   - 과전류 차단기, 접지, 피뢰설비

5. **3.5 가공인입선 및 배전선 공사**
   - 인입선, 전주, 애자

6. **3.6 고압 및 저압 배전반 공사**
   - 배전반, 분전반, 수변전설비

7. **3.7 특수장소 공사**
   - 위험물, 가연성 가스, 방폭공사

8. **3.8 전기응용시설 공사**
   - 조명, 동력, 제어배선

## 📊 Google Sheets 연동 (선택)

### 설정 방법
1. Google Sheets에서 스프레드시트 생성
2. **열 구성**: id, category, standard, detailItem, question, option1, option2, option3, option4, answer, explanation, imageUrl
3. 시트를 **공개**로 설정 (또는 API 키 설정)
4. Google Cloud Console에서 **Sheets API 활성화** 및 **API 키 생성**
5. `.env` 파일 생성:
```bash
VITE_GOOGLE_SHEETS_API_URL=https://your-apps-script-url
```
6. 관리자 페이지 → 동기화 탭에서 동기화 실행

### 동기화 기능
- **Google Sheets → LocalStorage**: 구글 시트 데이터를 로컬로 가져오기
- **LocalStorage → Google Sheets**: 로컬 데이터를 구글 시트로 백업
- **카테고리별 시트 관리**: 전기이론, 전기기기, 전기설비 시트별 분리
- **자동 ID 할당**: 중복 ID 자동 조정

## 🎯 사용 방법

### 1️⃣ 로그인
- **이름 입력**: 23명 중 선택 (예: 고광욱, 강형윤 등)
- **게스트 모드**: 기록 저장 안됨

### 2️⃣ 시험 모드 선택
- **랜덤 출제**: 정규 시험 (60문제)
- **카테고리별**: 집중 학습 (20문제)
- **오답 노트**: 복습 (최대 20문제)
- **이전 시험 계속하기**: 저장된 시험 이어서 풀기

### 3️⃣ 시험 응시
- 60분 카운트다운 타이머
- 문제 번호 클릭으로 이동
- 답안 선택 및 변경 가능
- **💾 저장하기** 버튼으로 수동 저장
- **📊 채점하기** 버튼으로 중간 점수 확인
- **🚪 나가기** 버튼으로 자동 저장 후 나가기
- 시험 제출 또는 자동 제출 (시간 종료 시)

### 4️⃣ 결과 확인
- 점수 및 합격/불합격 (60점 기준)
- 분야별 정답률
- 오답 자동 저장
- 해설 보기

### 5️⃣ 오답 노트
- 카테고리별 오답 분류 (색상 구분)
- 틀린 횟수, 연속 정답 횟수 표시
- **오답 복습하기**: 오답 문제만 선택해서 재응시
- 연속 3회 정답 시 자동 제거
