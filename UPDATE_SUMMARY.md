# 실전 모의고사 UI 업데이트 완료

## 변경된 사항

### ✅ 1. 답안 표기란 개선
- **문제번호**: 원형 → **사각 박스**로 변경
- **선택지 번호**: 크기 확대 (w-7 h-7 → **w-9 h-9**)
- **폰트**: text-xs → **text-sm font-bold**
- **시각적 강조**: shadow-md 추가

```
┌────────────────────────────────┐
│ 답안 표기란                      │
├────────────────────────────────┤
│ [1]  ① ② ③ ④                  │  ← 사각박스 + 큰 원형 번호
│ [2]  ① ② ③ ④                  │
│ [3]  ① ② ③ ④                  │
└────────────────────────────────┘
```

### ✅ 2. 화면 배치 단순화
- **이전**: 1단 / 2단 / 3단 선택 가능
- **변경**: **2단 레이아웃 고정** (문제 75% + 답안란 25%)
- UI 복잡도 감소, 일관된 사용자 경험 제공

### ✅ 3. 출제기준 표시 개선
이미지와 동일한 스타일로 변경:

```
문제 1  [전기이론]                [2.6 - 보호계전기]  선택: 2번
        └─ 연한 파랑 배경           └─ 파랑 배경, 흰 글자
           파랑 테두리
```

**스타일**:
- 카테고리: `bg-blue-50 text-blue-700 border border-blue-200`
- 출제기준: `bg-blue-600 text-white` (강조)
- 크기: `px-3 py-1` (동일한 패딩)

### ✅ 4. 선택지 동적 레이아웃
텍스트 길이에 따라 자동으로 배치 조정:

**짧은 텍스트 (평균 ≤ 25자)**
```
┌─────┬─────┬─────┬─────┐
│  ①  │  ②  │  ③  │  ④  │  ← 4줄 배치
└─────┴─────┴─────┴─────┘
```

**중간 텍스트 (25 < 평균 ≤ 50자)**
```
┌─────────────┬─────────────┐
│      ①      │      ②      │  ← 2줄 배치
├─────────────┼─────────────┤
│      ③      │      ④      │
└─────────────┴─────────────┘
```

**긴 텍스트 (평균 > 50자)**
```
┌─────────────────────────────┐
│              ①              │  ← 1줄 배치
├─────────────────────────────┤
│              ②              │
├─────────────────────────────┤
│              ③              │
├─────────────────────────────┤
│              ④              │
└─────────────────────────────┘
```

### ✅ 5. 제목 변경
- **이전**: "⚡ 전기기능사 CBT 모의고사"
- **변경**: "⚡ **전기기능사 CBT 실전 모의고사**"

### ✅ 6. 한 화면 최대 문제 수
- **기본값**: 3문제 (변경됨)
- **선택 가능**: 1문제 / 2문제 / 3문제
- 최대 3개 문제까지 한 화면에 표시

## 기술적 변경 사항

### src/pages/Exam.tsx

#### 1. Layout 고정
```typescript
// 이전
const [layout, setLayout] = useState<'single' | 'double' | 'triple'>('triple');

// 변경
// layout 변수 제거, 직접 w-3/4, w-1/4 사용
```

#### 2. 동적 그리드 레이아웃 로직
```typescript
// 선택지 텍스트 길이 계산 (평균)
const avgOptionLength = ([1, 2, 3, 4].reduce((sum, num) => {
  const optionKey = `option${num}` as keyof Question;
  return sum + (question[optionKey] as string).length;
}, 0) / 4);

// 길이에 따라 그리드 컬럼 결정
let gridCols = 'grid-cols-4'; // 4줄 (기본: 짧은 텍스트)
if (avgOptionLength > 50) {
  gridCols = 'grid-cols-1'; // 1줄 (긴 텍스트)
} else if (avgOptionLength > 25) {
  gridCols = 'grid-cols-2'; // 2줄 (중간 텍스트)
}
```

#### 3. 출제기준 박스 스타일

```typescript
  </span>
```


## UI 비교

### 문제 헤더
```
이전: 문제 1 / 60 [전기이론] [2.6] 선택: 2번

현재: 문제 1  [전기이론]    [2.6 - 보호계전기]  선택: 2번
      ↑       ↑ 연한배경       ↑ 파랑배경         ↑
      크게    테두리          강조됨             오른쪽정렬
```

### 선택지 번호
```
이전: (①) 32px × 32px, 일반 크기
현재: (①) 40px × 40px, 더 크고 굵게 (text-lg, font-bold)
```

### 답안 표기란
```
이전: (1) ① ② ③ ④  ← 모두 원형
현재: [1] ① ② ③ ④  ← 문제번호는 사각, 선택지는 큰 원형
```

## 사용자 경험 개선

1. **가독성 향상**
   - 큰 원형 번호로 선택지 구분 명확
   - 출제기준 강조로 학습 포인트 파악 용이

2. **효율적 공간 활용**
   - 짧은 선택지는 4줄로 컴팩트하게
   - 긴 선택지는 1줄로 여유있게

3. **일관된 UI**
   - 화면 배치 고정으로 혼란 감소
   - 실전 모의고사에 최적화된 레이아웃

4. **빠른 답안 표기**
   - 사각 박스로 문제번호 구분 명확
   - 큰 버튼으로 터치/클릭 정확도 향상
