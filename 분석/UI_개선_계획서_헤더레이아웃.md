# 📱 모바일 헤더 레이아웃 줄바꿈 현상 해결 계획서

## 1. 현상 분석
사용자가 제공한 스크린샷을 분석한 결과, 모바일 기기 또는 폭이 좁은 브라우저 창에서 다음과 같은 UI 깨짐 현상이 발생하고 있습니다.

1.  **제목 줄바꿈:** "전기기능사 CBT" 텍스트가 공간 부족으로 두 줄로 나뉘어 표시됨.
2.  **상태 텍스트 줄바꿈:** "(∞ 시간 제한 없음)" 텍스트가 좁은 영역에 갇혀 세로로 길게 늘어짐.
3.  **버튼 텍스트 깨짐:** "제보", "계산기", "저장하기", "나가기" 등의 버튼 텍스트가 강제로 줄바꿈되어("계\n산기", "저장\n하기") 버튼 모양이 찌그러짐.
4.  **원인:**
    *   화면 너비에 비해 표시하려는 콘텐츠(텍스트, 버튼)의 양이 많음.
    *   텍스트 줄바꿈을 방지하는 CSS 속성(`whitespace-nowrap`)이 적용되지 않음.
    *   모바일 환경에서 폰트 크기가 상대적으로 큼.
    *   버튼 내부의 텍스트가 모바일에서도 모두 노출되도록 설정되어 있음.

---

## 2. 해결 목표
*   모든 텍스트가 의도치 않게 줄바꿈되지 않도록 한다.
*   좁은 화면에서도 제목, 상태 정보, 버튼이 깔끔하게 정렬되도록 한다.
*   사용자 경험(UX)을 해치지 않는 선에서 정보를 축약하거나 숨긴다.

---

## 3. 해결 전략 (Step-by-Step)

### 전략 A: 텍스트 줄바꿈 방지 및 폰트 크기 최적화 (제목 영역)
제목과 상태 메시지가 한 줄, 혹은 자연스러운 두 줄로 배치되도록 스타일을 조정합니다.

1.  **`whitespace-nowrap` 적용:** 텍스트가 강제로 줄바꿈되지 않도록 설정합니다.
2.  **반응형 폰트 크기:** 모바일에서는 폰트 크기를 줄입니다.
    *   PC: `text-xl` (약 20px)
    *   모바일: `text-base` (약 16px) 또는 `text-sm` (약 14px)
3.  **Flex 구조 변경:** 제목과 상태 메시지를 가로(`row`) 배치가 아닌 세로(`col`) 배치로 변경하여 가로 공간 부족을 해결할 수도 있습니다. (선택 사항)

### 전략 B: 버튼 레이아웃 개선 (버튼 영역)
버튼이 찌그러지는 것을 막기 위해 버튼의 스타일을 개선합니다.

1.  **텍스트 줄바꿈 금지:** 버튼 텍스트에 `whitespace-nowrap`을 적용합니다.
2.  **모바일 텍스트 숨김 (권장):** 공간이 매우 좁을 경우, 아이콘만 표시하고 텍스트는 숨깁니다.
    *   예: `<span>💾</span> <span className="hidden sm:inline">저장하기</span>`
3.  **Grid 레이아웃 도입:** Flex 대신 Grid를 사용하여 버튼들이 좁은 공간에서도 일정한 비율로 정렬되도록 합니다.
4.  **텍스트 축약:** "저장하기" -> "저장", "나가기" -> "종료" 등으로 글자 수를 줄입니다.

## 4. 구체적인 실행 계획 (Code Implementation)

`src/pages/Exam.tsx` 파일을 다음과 같이 수정할 예정입니다.

### 4.1. 제목 및 상태 텍스트 수정
```tsx
{/* 수정 전 */}
<div className="flex items-center gap-3">
  <h1 className="text-xl font-bold text-gray-800">⚡ 전기기능사 CBT</h1>
  {/* ... */}
</div>

{/* 수정 후 (제안) */}
<div className="flex flex-col sm:flex-row sm:items-center gap-1 sm:gap-3">
  <h1 className="text-base sm:text-xl font-bold text-gray-800 whitespace-nowrap">
    ⚡ 전기기능사 CBT
  </h1>
  {/* 상태 텍스트도 모바일에서는 작게 표시 */}
  <span className="text-xs sm:text-sm font-bold text-green-600 whitespace-nowrap">
    (∞ 시간 제한 없음)
  </span>
</div>
```

### 4.2. 버튼 영역 수정
버튼 내부의 텍스트를 모바일(`sm` 미만)에서는 숨기거나 축약합니다.

```tsx
{/* 수정 전 */}
<button ...>
  📋 제보
</button>

{/* 수정 후 (제안) */}
<button ... className="... flex items-center justify-center gap-1">
  <span className="text-lg">📋</span>
  {/* 모바일에서는 텍스트 숨김, sm 이상에서만 표시 */}
  <span className="hidden sm:inline text-sm">제보</span>
</button>
```

### 4.3. 전체 컨테이너 여백 조정
모바일에서 패딩(`p-3`)을 줄여 콘텐츠 공간을 확보합니다.
```tsx
<div className="bg-white rounded-lg shadow-md p-2 sm:p-3 mb-2">
```
