# 시험 모드별 레이아웃 분리 완료 ✅

## 변경 사항

실전 모의고사와 랜덤 60문제가 **완전히 다른 레이아웃**을 사용하도록 수정했습니다.

## 모드별 레이아웃

### 1. 🎯 실전 모의고사 (timedRandom)
**새로운 컴팩트 레이아웃 - 4문제 동시 표시**

```
┌────────────────────────────────────────────────────────┐
│ ⚡ 전기기능사 CBT 실전 모의고사                          │
├────────────────────────────────────────────────────────┤
│ [글자크기: 100% 150% 200%] [문제수/화면: 1 2 3 ④]      │
├─────────────────────────────────┬──────────────────────┤
│ 문제 1  [전기기기]               │ 답안 표기란          │
│ [2.6 - 보호계전기]              │ [1]  ① ② ③ ④       │
│ ...선택지 (컴팩트)               │ [2]  ① ② ③ ④       │
│ ─────────────────────           │ [3]  ① ② ③ ④       │
│ 문제 2  [전기이론]               │ ...                  │
│ [1.5 - 교류회로]                │ [60] ① ② ③ ④       │
│ ...선택지 (컴팩트)               │                      │
│ ─────────────────────           │                      │
│ 문제 3                          │                      │
│ 문제 4                          │                      │
└─────────────────────────────────┴──────────────────────┘
│ [← 이전 페이지]  페이지 1/15  [다음 페이지 →] [📝 제출] │
└────────────────────────────────────────────────────────┘
```

**특징:**
- ✅ 한 화면에 최대 4문제 표시
- ✅ 문제수/화면 선택 가능 (1~4)
- ✅ 컴팩트한 선택지 (박스 없음)
- ✅ 페이지 단위 네비게이션
- ✅ 답안 표기란 (최소 너비)
- ✅ 세부 출제기준 표시

### 2. 🎲 랜덤 60문제 (untimedRandom)
**예전 스타일 - 큰 카드 레이아웃**

```
┌────────────────────────────────────────────────────────┐
│ ⚡ 전기기능사 CBT 실전 모의고사                          │
├────────────────────────────────────────────────────────┤
│ [글자크기: 100% 150% 200%]                              │
├─────────────────────────────────┬──────────────────────┤
│                                  │ 답안 표기란          │
│  ┌──────────────────────────┐  │ [1]  ① ② ③ ④       │
│  │ 문제 1   [선택: 2번]      │  │ [2]  ① ② ③ ④       │
│  │ [전기기기]                │  │ [3]  ① ② ③ ④       │
│  │ [2.3 - 유도전동기]        │  │ ...                  │
│  ├──────────────────────────┤  │ [60] ① ② ③ ④       │
│  │                          │  │                      │
│  │ [문제 텍스트 영역]        │  │                      │
│  │  - 큰 폰트              │  │                      │
│  │  - 여유로운 간격        │  │                      │
│  │                          │  │                      │
│  ├──────────────────────────┤  │                      │
│  │                          │  │                      │
│  │ 1. 선택지 1 (큰 박스)    │  │                      │
│  │                          │  │                      │
│  │ 2. 선택지 2 (큰 박스)    │  │                      │
│  │                          │  │                      │
│  │ 3. 선택지 3 (큰 박스)    │  │                      │
│  │                          │  │                      │
│  │ 4. 선택지 4 (큰 박스)    │  │                      │
│  │                          │  │                      │
│  └──────────────────────────┘  │                      │
│                                  │                      │
└─────────────────────────────────┴──────────────────────┘
│ [← 이전]  1 / 60  [다음 →]  [📝 제출]                   │
└────────────────────────────────────────────────────────┘
```

**특징:**
- ✅ 한 화면에 1문제만 표시
- ✅ 큰 카드 스타일 (shadow-lg)
- ✅ 선택지에 큰 박스 (border-2)
- ✅ 여유로운 간격 (p-5, p-6, p-8)
- ✅ 문제 단위 네비게이션 (이전/다음)
- ✅ 답안 표기란 동일

### 3. 📚 카테고리별 / 📝 오답노트
**랜덤 60문제와 동일한 레이아웃 사용**

## 기술적 구현

### 1. 초기값 설정
```typescript
const [questionsPerPage, setQuestionsPerPage] = useState<1 | 2 | 3 | 4>(
  examMode === 'timedRandom' ? 4 : 1  // 모드별 기본값
);
```

### 2. 조건부 렌더링
```typescript
{examMode === 'timedRandom' ? (
  // 실전 모의고사: 컴팩트 다중 문제 레이아웃
  <div className="space-y-8">
    {currentPageQuestions.map(...)}
  </div>
) : (
  // 랜덤 60문제: 큰 카드 단일 문제 레이아웃
  <div className="bg-white rounded-lg shadow-lg p-8 max-w-4xl mx-auto">
    ...
  </div>
)}
```

### 3. UI 컨트롤 표시 제어
```typescript
{/* 실전 모의고사에만 문제수 선택 버튼 표시 */}
{examMode === 'timedRandom' && (
  <div className="flex items-center gap-2">
    <button>1문제</button>
    <button>2문제</button>
    <button>3문제</button>
    <button>4문제</button>
  </div>
)}
```

### 4. 네비게이션 분기
```typescript
{examMode === 'timedRandom' ? (
  // 페이지 단위
  <button>← 이전 페이지</button>
  <button>다음 페이지 →</button>
) : (
  // 문제 단위
  <button>← 이전</button>
  <button>다음 →</button>
)}
```

## 스타일 비교표

| 요소 | 실전 모의고사 | 랜덤 60문제 |
|-----|-------------|-----------|
| **문제 수/화면** | 1~4 (선택 가능) | 1 (고정) |
| **카드 스타일** | 없음 | `shadow-lg` |
| **선택지 박스** | 없음 | `border-2` |
| **선택지 패딩** | `p-2` | `p-5` |
| **문제 패딩** | 최소 | `p-6, p-8` |
| **폰트 크기** | 표준 | 큰 편 (`text-xl`) |
| **네비게이션** | 페이지 단위 | 문제 단위 |
| **문제수 버튼** | 표시 | 숨김 |

## 사용자 경험

### 실전 모의고사
- **목적**: 실제 시험과 유사한 환경
- **장점**: 빠른 문제 확인, 효율적인 시간 관리
- **UI**: 컴팩트하고 정보 밀도 높음

### 랜덤 60문제
- **목적**: 편안한 학습, 문제 집중
- **장점**: 큰 화면으로 문제 명확히 확인
- **UI**: 여유롭고 가독성 높음
