# PDF 파일 일괄 Import 방법

## 📂 자동 Import 완료!

PDF 파싱 시스템이 성공적으로 구현되었습니다. 이제 브라우저에서 간단하게 모든 PDF 파일을 import할 수 있습니다.

## 🚀 Import 방법 (3분 소요)

### 1단계: 브라우저에서 관리자 페이지 열기
```
http://localhost:5174/admin
```

### 2단계: 로그인
- 비밀번호 입력: `admin2024`

### 3단계: "📂 일괄 Import" 탭 클릭

### 4단계: PDF 파일들 선택
1. "PDF 파일을 선택하거나 드래그하세요" 영역 클릭
2. 파일 선택 대화상자에서 `d:\cbt\electrical-cbt\source` 폴더로 이동
3. **Ctrl+A로 모든 PDF 파일 선택** (또는 원하는 파일들만 Shift+클릭으로 선택)
4. "열기" 버튼 클릭

### 5단계: 자동 파싱 대기
- PDF.js가 자동으로 모든 파일을 파싱합니다
- 진행 상황이 화면에 표시됩니다
- 문제 형식이 맞는 파일들만 자동으로 추출됩니다

### 6단계: 결과 확인 및 수정
- 파싱된 문제 목록이 테이블로 표시됩니다
- 각 문제의 카테고리가 자동 분류됩니다:
  - ✅ **전기이론** / **전기기기** / **전기설비** - 자동 분류 완료
  - ⚠️ **미분류** - 수동으로 카테고리 선택 필요 (드롭다운 클릭)
- 정답도 수정 가능합니다 (드롭다운 클릭)

### 7단계: 일괄 Import
- 모든 내용 확인 후 **"💾 일괄 Import"** 버튼 클릭
- LocalStorage에 저장됩니다

## ✅ 완료!

이제 "📝 문제 관리" 탭에서 import된 모든 문제를 확인할 수 있습니다.

## 📊 예상 결과

- **처리 파일 수**: 60개 PDF 파일
- **추출 문제 수**: 약 1,000+ 문제 (파일 형식에 따라 다름)
- **소요 시간**: 약 2-5분 (파일 크기와 수량에 따라 다름)

## 🔍 주요 기능

### 1. 자동 카테고리 분류
키워드 기반으로 3과목 자동 분류:
- **전기이론**: 옴의 법칙, 키르히호프, 전압, 전류, 저항, 콘덴서 등
- **전기기기**: 변압기, 발전기, 전동기, 동기기, 유도기 등
- **전기설비**: 배선, 접지, 차단기, 개폐기, 퓨즈, 배전 등

### 2. 자동 정답 추출
- 각 PDF의 정답 키 페이지에서 자동으로 정답 추출
- 정답 형식: 1.가 2.나 3.다... 패턴 인식

### 3. 이미지 감지
- 이미지가 있는 문제는 "✅" 표시
- 나중에 이미지 파일을 별도로 업로드 가능

### 4. 편집 기능
- 파싱 후 모든 필드 수정 가능
- 카테고리와 정답은 드롭다운으로 쉽게 변경

## ⚠️ 주의사항

1. **파일 형식**: 문제 형식이 일정하지 않은 PDF는 자동 파싱이 실패할 수 있습니다
2. **정답 확인**: 자동 추출된 정답이 정확한지 반드시 확인하세요
3. **미분류 문제**: "미분류"로 표시된 문제는 수동으로 카테고리를 지정하세요
4. **이미지**: 이미지가 필요한 문제는 별도로 이미지 파일을 준비하세요

## 🎯 팁

- **대용량 처리**: 파일이 많으면 여러 번에 나눠서 import하세요 (한 번에 10-20개씩)
- **미리보기 활용**: Import 전에 미리보기에서 문제들을 검토하세요
- **백업**: Import 전에 기존 데이터를 백업하세요 (LocalStorage 내보내기)

## 🆘 문제 해결

### Q: PDF 파일이 선택되지 않아요
A: 파일 선택 대화상자에서 파일 형식을 "모든 파일 (*.*)" 또는 "PDF (*.pdf)"로 설정하세요

### Q: 파싱이 실패했어요
A:
1. 해당 PDF 파일의 형식을 확인하세요
2. 문제가 텍스트로 되어 있는지 확인하세요 (이미지 스캔본은 파싱 불가)
3. 브라우저 콘솔(F12)에서 에러 메시지를 확인하세요

### Q: 문제 수가 예상보다 적어요
A: 문제 형식이 일정하지 않은 파일은 자동 파싱이 실패합니다. 해당 파일들은 수동으로 입력하거나 형식을 확인하세요

## 📞 추가 지원

문제가 발생하면 브라우저 콘솔(F12)을 열어 에러 메시지를 확인하세요.
