# 실전모의 평가 - 다중 문제 표시 기능 업데이트

## 변경 사항 요약

실전모의 평가 화면을 재구성하여 한 화면에 2개 이상의 문제를 동시에 표시할 수 있도록 개선했습니다.

## 주요 개선 사항

### 1. 문제수/화면 선택 기능 추가

- **1문제**: 기존처럼 한 화면에 한 문제만 표시
- **2문제**: 한 화면에 2개 문제 표시 (기본값)
- **3문제**: 한 화면에 3개 문제 표시

사용자가 자신의 학습 스타일과 화면 크기에 맞춰 선택할 수 있습니다.

### 2. 페이지 기반 네비게이션

기존의 문제 단위 네비게이션을 페이지 단위로 변경:
- **이전/다음 버튼**: 페이지 단위로 이동
- **페이지 표시**: "페이지 1/30 (문제 1-2 / 60)" 형식으로 현재 위치 명확하게 표시
- **답안 표기란**: 전체 문제 목록에서 빠른 이동 가능

### 3. 개선된 UI/UX

#### 문제 영역
- 각 문제는 구분선으로 명확히 분리
- 문제 번호, 카테고리, 출제 기준 표시
- 선택한 답안을 실시간으로 표시
- 스크롤을 통해 한 페이지의 모든 문제 확인 가능

#### 툴바
```
[글자크기: 100% 150% 200%] [문제수/화면: 1문제 2문제 3문제] [화면배치: 1단 2단 3단]
```

#### 하단 네비게이션
```
[🔢 계산기] [← 나가기]     페이지 1/30 (문제 1-2 / 60)     [← 이전 페이지] [다음 페이지 →]     [📝 답안 제출]
```

### 4. 향상된 학습 효율

- **빠른 문제 비교**: 유사한 문제를 동시에 보며 패턴 파악 가능
- **효율적인 시간 관리**: 페이지 단위로 이동하여 시간 절약
- **유연한 학습 방식**: 개인의 선호도에 맞게 화면 구성 조정

## 사용자 가이드

### 실전모의 평가 시작하기

1. 홈 화면에서 "🎯 실전 모의고사 (60분 제한)" 선택
2. "🚀 시험 시작" 클릭

### 화면 설정 조정하기

**글자 크기**
- 작은 글자를 선호하면: 100%
- 표준 크기: 150%
- 큰 글자를 선호하면: 200%

**문제수/화면**
- 한 문제씩 집중하고 싶다면: 1문제
- 균형잡힌 학습 (권장): 2문제 ✓
- 빠른 문제 훑기: 3문제

**화면배치**
- 문제만 크게 보기: 1단
- 문제 + 답안란: 2단
- 문제 + 답안란 (넓게): 3단 ✓

### 문제 풀이하기

1. 스크롤하여 현재 페이지의 모든 문제 확인
2. 각 문제의 선택지를 클릭하여 답안 선택
3. "다음 페이지 →" 버튼으로 다음 문제들로 이동
4. 우측 답안 표기란에서 전체 진행 상황 확인
5. 답안 표기란의 문제 번호 클릭으로 특정 문제로 빠르게 이동

### 답안 제출하기

1. 모든 문제를 풀었는지 확인 (우측 상단 "안 푼 문제 수" 확인)
2. "📝 답안 제출" 클릭
3. 확인 메시지에서 "확인" 클릭
